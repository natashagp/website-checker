# Website Checker

### Step 1

Clone the repository

```shell
git clone https://github.com/natashagp/website-checker.git
```

### Step 2

Change the directory to website-checker

```shell
cd website-checker
```

### Step 3

Install requirements

```shell
pip install -r requirements.txt
```

### Step 4

Run project

```shell
python main.py
```
